# Latijn H16 - Quiz App

Quiz PWA voor Latijn Grammatica les 16. Offline beschikbaar op elke telefoon.

**Live:** https://karel314.github.io/latijn-h16/

## Inhoud

- **125 vragen** over 7 leerdoelen
- **Vocabulaire H16** — exemplum, vincere, interficere, tradere, abesse, tantus, nisi, tuus, -que
- **Aanwijzende voornaamwoorden** — hic/haec/hoc, ille/illa/illud, iste/ista/istud, is/ea/id
- **Verbuiging** — alle naamvallen en geslachten
- **Vormherkenning & vormproductie** — naamval bepalen, juiste vorm invullen
- **Vertaling & toepassing** — zinsvertaling Latijn → Nederlands, aanwijzend vs persoonlijk
- **Bijwoorden** — groep 1 en 2, bijvoeglijk vs bijwoordelijk

## Features

- 3 vraagtypes: meerkeuze, invullen, meerdere antwoorden
- Woordkaartjes: 13 Latijn → Nederlands flashcards
- Flashcards per topic (alle of alleen zwakke topics)
- 7 badges te verdienen bij 80% beheersing
- Adaptive herhaling: zwakke vragen komen vaker terug
- Offline beschikbaar als PWA (installeerbaar op telefoon)
- Voortgang opgeslagen in localStorage

## Technologie

Vanilla HTML/CSS/JS, geen frameworks. Service worker voor offline caching. Gehost via GitHub Pages.
